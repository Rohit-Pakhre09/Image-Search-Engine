<h1>📷 Unsplash Image Search App</h1>

<p>
  This is a dynamic web application that allows users to search and view stunning images directly from the 
  <a href="https://unsplash.com/developers" target="_blank">Unsplash API</a>. 
  The project uses modern web technologies to deliver a clean, responsive, and interactive user experience.
</p>

<img src="https://github.com/Rohit-Pakhre09/Image-Search-Engine/blob/a54310a3900f886dfc351638ca2dfcfc8243dd5c/Image%20Search%20Engine/Assets/Image%20Search%20Engine.png" alt="Project">

<h2>🚀 Features</h2>
<ul>
  <li>🔍 Search for high-quality images using Unsplash API.</li>
  <li>📸 Dynamically displays image results based on user queries.</li>
  <li>💡 Displays image descriptions using smooth overlays.</li>
  <li>📱 Fully responsive design that works on all screen sizes (mobile, tablet, desktop).</li>
  <li>🎨 Clean and modern UI with intuitive user interaction.</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li>HTML5, CSS3, and JavaScript (ES6+)</li>
  <li>Unsplash RESTful API</li>
  <li>Responsive layout using Grid-Box and Media Queries</li>
</ul>
